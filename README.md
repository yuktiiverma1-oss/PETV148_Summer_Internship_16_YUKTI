# Basic Banner Grabbing Tool

## 📌 Project Overview
The Basic Banner Grabbing Tool is a Flask-based web application developed using Python. It allows users to retrieve banner information from network services running on specified ports. Banner grabbing helps identify services, software versions, and protocols running on a target system.

## 🚀 Features
- User Login & Registration
- Scan one or more ports
- Retrieve service banners
- Simple and responsive web interface
- Download scan reports
- Built using Python Flask

## 🛠 Technologies Used
- Python 3
- Flask
- Socket Programming
- HTML
- CSS
- Bootstrap

## ⚙ Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/PETV148_summer_internship_16_yukti.git
```

2. Move into the project folder

```bash
cd PETV148_summer_internship_16_yukti
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
